# RTAPS - Real-Time Adaptive Procedure System

A web-based procedure management system for tracking and analyzing step-by-step processes.

## Live Demo

**Live Site**: [https://d1789m780yh1c4.cloudfront.net](https://d1789m780yh1c4.cloudfront.net) 🔒

**Legacy HTTP Site**: [http://rtaps-app.s3-website.us-east-2.amazonaws.com/](http://rtaps-app.s3-website.us-east-2.amazonaws.com/) (not secure)

## Features

- **User Management**: Participant registration and login
- **Procedure Sessions**: Step-by-step process tracking
- **Analytics Dashboard**: Session data and performance metrics
- **Admin Panel**: User management and system overview
- **Responsive Design**: Works on desktop and mobile

## Quick Start

1. Visit the live site
2. Register as a new participant or login
3. Start a procedure session
4. View analytics in the dashboard

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Procedures Available

- Centrifuge Operations
- Column Flushing
- Pressure Management
